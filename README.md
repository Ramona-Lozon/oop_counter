## oop_counter

By Ramona Lozon

for Marco DeLuca

The purpose of this assignment is to create a site with an operation "counter", the counter will feature 3 buttons, 1 for increasing the count by 1 incrementally, 1 for reducing the count by 1 incrementally, and 1 button for resetting the counter to its starting number. One caveat is that the number can never fall below 0.

# dependancies
This program utilizes HTML5, CSS3+SASS, and JavaScript organized into modules. 

# Expectation
The deliverables will be divided into 3 due dates, each due date will require the project to reach a certain level of completion. 

# installation
simply download onto your machine, extract the ZIP file contents and open index.html using any browser.
