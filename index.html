Increment Counter using Vue

This website is a demonstration of basic Vue.js principles, demonstrated through the use of increment counters

The Way the Counter works

the three counters below will increment upwards and downwards when the associated button is clicked, all three will increment a different amount. when a counter reaches 0, it will no longer decrement and the reset and decrement buttons will be disabled.

The Code Behind it

Mounting the methods

The counters are coded using the Vue.js framework, they are injected using Vue and defined within the counter class in "js/modules/counter.js". The counter class is imported into "js/main.js" where it is associated to an HTML element with the id of "counter-container#", from there the class "Mounts" several elements made within counter.js to the index page. the Mount method defined the counter itself, the initial state of the counter, and defines the buttons within the counter's container.

How the counters increment

The way the counters increment, decrement and reset are defined within the methods that follow the initial mounting of the counter class.
