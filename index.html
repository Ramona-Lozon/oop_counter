Increment Counter using Vue

This website is a demonstration of basic Vue.js principles, demonstrated through the use of increment counters

The Way the Counter works

the three counters below will increment upwards and downwards when the associated button is clicked, all three will increment a different amount. when a counter reaches 0, it will no longer decrement and the reset and decrement buttons will be disabled.

The Code Behind it

The Counter Class

the class is initially defined using a "constructor" sets up a given class' properties, and defines the structure of the syntax of the class. for example;

js/modules/counter.js
export class Counter { constructor(value, value){}
}

this will be at the top of most classes, and will define how the class will connect to the front end.

for this class our constructor will looks like this;

constructor(selector, initialValue = 0){}

the "selector" is where we will put that value that connects it to an object in the HTML, which is most often a HTML element's ID.

"initialValue = 0" is defining that the next value within the string will define the initial value of whatever the class contains, and its default is equal to 0.

the line above the constructor is an export function, as this class will be exported. within "js/main.js" the class will be imported like so;

import { Counter } from './counter.js';

after theclass has been imported, we can create an instance that will define how the class will be used;

const counter1 = new counter('#objectId', 1);

in this line, "const" creates a new instance,
"counter1" gives that instance a name,
"new counter" creates a new instance of the counter class, and takes the place of the "constructor",
the text in brackets afterwards is equal to the values entered into the constructor, like so;

constructor(selector, initialValue=0){}
new counter('#objectId', 1)

the two strings match each other, selector is how the class is added to an object, in this case the object would have an ID of "#objectID" and the second value in the new counter instance would override the "initialValue" setting it to 1.

How the counters increment

The way the first counter increments, decrements and resets are defined within the methods that follow the initial mounting of the counter class.

Incrementing is done by using an incrememnt method that contains "this.count++;" within this line of code, "this" refers to the instance of the class being used, "count" is a special property that stores the current vlaue of the counter.

the initial value of the counter is handled at the very top of the class by the "constructor" this method defines the initial value of the counter.

the full function would look like this;

export class Counter { constructor(selector, initialValue=0) {
this.count = initialValue;
this.selector = selector;
this.mount();
}
}
